# 1688 选品 → 店小秘上架流程（2026-06-11）

## 前置说明

Hermes 在 WSL 环境中无法直接访问 1688（JS 渲染 + 反爬），因此 1688 选品由用户在浏览器中操作，Agent 负责数据分析和上架方案。

## 完整工作流

```
用户操作（浏览器） → Agent 分析 → 用户操作（店小秘） → Agent 指导
```

### 第一步：用户采集 1688 产品

1. 用户打开 1688.com，搜索目标产品关键词（如"宠物折叠硅胶碗"）
2. 按销量排序，挑 3-6 个价格/款式合适的
3. 通过店小秘「采集」功能将产品导入店小秘采集箱
4. 完成后通知 Agent

### 第二步：Agent 产出上架方案

Agent 基于以下数据源生成方案：

| 数据来源 | 用途 |
|:--|:--|
| 行业市场报告（Exploding Topics/APPA） | 搜索趋势、增长率、市场容量 |
| 速卖通竞品分析数据 | 价格分布、畅销功能、用户痛点 |
| 1688 产品标题 & 参数（用户提供） | 选品依据、成本参考 |
| 已知类目属性映射表 | 系统属性选值 |

输出内容：
- 产品标题（英文 + 目标国家语言）
- 7 个 SKU 规格规划
- 定价策略（引流款/利润款/形象款）
- 主图方案
- 卖点提炼
- 差评预防措施

### 第三步：用户将产品移入待发布

用户操作：
1. 在店小秘采集箱找到产品
2. 点击「移入待发布」
3. 弹窗点「确 定」（文字中间有空格）
4. 在待发布列表点「编辑」打开编辑页

### 第四步：Agent 指导填表

Agent 逐项指导用户填写（不 CDP 操控，仅指挥）：

| 顺序 | 项目 | 说明 |
|:--:|:--|:--|
| 1 | 标题 | 用户粘贴 Agent 给的标题 |
| 2 | 品牌 | 选 NONE |
| 3 | 其他系统属性 | 按属性映射表选 |
| 4 | 产品图片 | 用户上传 6 张主图 |
| 5 | SKU 配置 | 按 SKU 表填颜色/价格/库存 |
| 6 | 包装信息 | 重量 + 尺寸 |
| 7 | 运费/服务模板 | 下拉选择 |
| 8 | 描述信息 | 用新版编辑器一键生成 |
| 9 | 保存并移入待发布 | 最后一步 |
