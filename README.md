## ⚡ 一句话

把 1688 商品一键搬到速卖通。AI 替你操作店小秘 ERP，填属性、定价、传图、写描述、提交发布。

---

## 🤔 解决什么问题

跨境卖家每天花 2-3 小时手动填店小秘表单——标题翻译、属性匹配、SKU 定价、图片上传、描述编写。

这个 skill 让 AI 替你操作浏览器，全流程自动化。
